HahuWeb

Introduction

Welcome. Hahu CMS is a web based content management system, allows users to create awesome website. This document describe haw to install hahu CMS manually.

Installation

  1. Unzip the package and move to webserver directory.
  2. Create database that you used on Hahu CMS.
  3. Open hw-admin/ in your browser. It will take you through the process to set up a config.php file with your database connection details.
    • If for some reason this does not work, Open up hw-includes/config/config-sample.php with a text editor like WordPad or similar and fill in your database connection details.
    • Save the file as config.php and upload it.
    • Open hw-admin/ in your browser.
  4. Once the configuration file is set up, the installer will set up the tables needed for your site. If there is an error, double check your config.php file, and try again. If it fails again.
  5. The installer should then send you to the login page.

Create Account

Enter your username, email and password and click create. It will send you to home page.

Updating Manually

  1. Before you update anything, make sure you have backup copies of any files.
  2. Download from hahuweb website
  3. Go to setting under version click update button and update.

System Requirements

  • PHP version 5.6.20 or greater.
  • MySQL version 5.0 or greater.

Recommendations

  • PHP version 7.4 or greater.
  • MySQL version 5.7 or greater OR MariaDB version 10.2 or greater.
